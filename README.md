# paintjs
Painting Board made with VanilaJS

# 배운 내용
- Canvas API
  - beginPath(), moveTo(), lineTo(), stroke
  - fillStyle, fillRect
  - strokeStyle, lineWidth
  - toDataURL()
- mouse 이벤트 (mousedown, mouseup, mouseleave, mousemove)
- contextmenu 이벤트
- event.preventDefault()
- a 태그의 href와 download 애트리뷰트
